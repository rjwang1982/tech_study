## 📖 简介

这是一份完整的 AWS 技术支持工程师学习路径文档，旨在帮助技术支持工程师系统化地学习 AWS 云服务。

## 📚 内容概览

- **阶段一：基础技能** - Linux、网络、数据库基础
- **阶段二：AWS 核心服务与部署** - 计算、存储、网络、数据库、监控、云部署
- **阶段三：专业领域进阶** - 云迁移、云安全、云灾备
- **阶段四：高级实践** - 架构设计、成本优化、自动化运维、容器、大数据、机器学习
- **实战项目** - 5 个完整的实战项目
- **学习资源** - 精选的官方文档、书籍、社区资源

## 🎯 学习理念

本学习路径基于"知行合一"的理念：

1. **临渊羡鱼，不如退而结网** - 实践是唯一的捷径
2. **以终为始** - 用项目驱动学习
3. **最好的学习是"教"** - 输出倒逼输入
